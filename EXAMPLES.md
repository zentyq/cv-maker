# Usage Examples

This file contains example job listings you can use to test the CV builder.

## Example 1: Full Job Listing (Software Engineer)

```
Senior Software Engineer - Full Stack

We are seeking an experienced Senior Software Engineer to join our growing team. 

Responsibilities:
- Design and implement scalable web applications using React and Node.js
- Lead technical architecture decisions and mentor junior developers
- Collaborate with product managers to translate requirements into technical solutions
- Implement automated testing and CI/CD pipelines
- Optimize application performance and database queries

Requirements:
- 5+ years of professional software development experience
- Strong proficiency in JavaScript, TypeScript, React, and Node.js
- Experience with cloud platforms (AWS, Azure, or GCP)
- Knowledge of SQL and NoSQL databases
- Excellent problem-solving and communication skills
- BS in Computer Science or equivalent experience

Nice to Have:
- Experience with microservices architecture
- Knowledge of containerization (Docker, Kubernetes)
- Previous leadership or mentoring experience
```

## Tips for Best Results

### For Generic CVs
Just enter a job title:
- "Software Engineer"
- "Project Manager"
- "Data Analyst"

The AI will generate a general CV appropriate for that role.

### For Targeted CVs
Paste the full job description including:
- Responsibilities
- Required skills
- Nice-to-have qualifications
- Company description

The AI will tailor the CV specifically to that job posting.

### For Specific Industries
Add industry context:
- "Healthcare Data Analyst"
- "Fintech Product Manager"
- "E-commerce Marketing Specialist"

This helps the AI generate industry-relevant experience.

## Testing Different Templates

After generating a CV, try switching between templates to see which layout works best:

1. **Simple ATS** - Best for:
   - Applicant Tracking Systems
   - Conservative industries (finance, healthcare)
   - When keyword matching is critical

2. **Modern** - Best for:
   - Tech companies
   - Startups
   - General professional use

3. **Two-Column** - Best for:
   - When you have lots of skills to display
   - Design-conscious companies
   - Maximizing space efficiency

4. **Creative** - Best for:
   - Creative industries (marketing, design)
   - Startups with modern culture
   - Making a strong visual impression

## Advanced Usage

### Multiple Versions
Generate different versions for different applications:
1. Generate CV for "Senior Developer"
2. Download PDF as "CV-Technical.pdf"
3. Paste new job listing for "Team Lead Developer"
4. Generate new CV
5. Download as "CV-Leadership.pdf"

### Iterative Refinement
1. Start with just a job title
2. Review the generated CV
3. Add more specific requirements or skills to the job listing
4. Regenerate to see more targeted results

### Custom Data
If you want to test the templates with specific data:
1. Generate any CV first
2. Open browser dev console
3. Modify the cvData state
4. The preview will update automatically
